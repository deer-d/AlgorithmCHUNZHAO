        delete: O(n)

        链表：
        prepend: O(1)
        append: O(1)
        lookup: O(n)
        insert: O(1)
        delete: O(1)

        跳表：
        要是有序的
        对标平衡树，二分查找
        lookup: O(logn)
        insert: O(logn)
        delete: O(logn)

        空间复杂度O（n）
